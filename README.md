# Tramper Pay 💷

Weekly net pay calculator for tramper (long-haul) truck drivers in the UK.

Built for real-world use on mobile — works offline, installs as an app.

## Features

- Weekly net pay breakdown — day by day
- Supports weekdays, Saturdays, Sundays, Bank Holidays and holidays
- Automatic overtime calculation (over 11 hours on weekdays)
- Bonuses: ADR, Genset, Splitter
- Samsara driving score bonus (Band A / B / C)
- Night Out and Night In allowances
- Bilingual UI — Ukrainian 🇺🇦 and English 🇬🇧
- Generates a downloadable weekly report (.txt) to send to your accountant
- All data saved automatically — survives page refresh
- Works offline (PWA — installable from Chrome on Android)

## Live App

👉 [https://nazaracc.github.io/tramper-pay/](https://nazaracc.github.io/tramper-pay/)

Open in Chrome on your phone → tap the menu → **Install app** or **Add to Home Screen**.

## How to Use

1. Set the week start date (Sunday)
2. Enter your name and truck number
3. For each day — select day type, enter hours worked, add any bonuses
4. Net pay updates instantly
5. Tap **Weekly Report** to download a .txt summary for your accountant

## Tech

Plain HTML + CSS + JavaScript. No frameworks, no dependencies. PWA-ready with offline support via Service Worker.
